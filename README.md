# SIBO

This Python script is to find a non-SIBO matroid.

## Dependencies
- [poetry](https://python-poetry.org/)

## How to run
### Instal dependencies
```sh
poetry install
```

### Run
```sh
poetry run python sibo.py -r <RANK>
```

Designate the rank of the matroid you want to find as `<RANK>`.
